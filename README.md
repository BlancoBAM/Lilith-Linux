### Custom AI Models

#### Adding New Models
```bash
# 1. Download quantized model
cd /opt/lilith-ai/models
wget https://huggingface.co/.../your-model.gguf

# 2. Create custom task configuration
nano /opt/lilith-ai/config/tasks/custom.json

# 3. Register with AI system
lilith-admin add-model "Custom Assistant" \
  --model /opt/lilith-ai/models/your-model.gguf \
  --config /opt/lilith-ai/config/tasks/custom.json
```

#### Fine-tuning Existing Models
```bash
# Generate training data
lilith-admin export-conversations > training-data.jsonl

# Fine-tune model
lilith-admin fine-tune \
  --base-model llama-7b-q4.gguf \
  --training-data training-data.jsonl \
  --output custom-assistant.gguf
```

## ğŸ› Troubleshooting

### Build Issues

**Source integration fails:**
```bash
# Check disk space
df -h /opt

# Clean and retry
sudo rm -rf /opt/lilith-sources
lilith-source-integration.sh
```

**AI model download fails:**
```bash
# Check network connectivity
ping huggingface.co

# Manual download if needed
cd /opt/lilith-ai/models
wget [model-url] -O main-model.gguf
```

### Performance Issues

**AI responses slow:**
- Reduce model size (7B â†’ 3B)
- Use more aggressive quantization (Q4_K_M â†’ Q2_K)
- Decrease context length (2048 â†’ 1024)
- Check CPU thermal limits

**High memory usage:**
```bash
# Monitor AI memory usage
lilith-admin memory-profile

# Adjust quantization

```

### Integration Problems
